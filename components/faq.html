Frequently Asked Questions

Got questions? We've got answers.

What is ReactStream?

ReactStream is a comprehensive CLI toolkit designed for React component development. It provides tools for analyzing components, detecting issues, and testing components in isolation without complex setup.

Do I need to change my existing React project to use ReactStream?

No! ReactStream works with any React project and doesn't require any special configuration or changes to your codebase. Just install it and start analyzing or testing your components.

Can ReactStream fix issues automatically?

Yes, by using the --fix option with the analyze command, ReactStream can automatically fix many common issues like unused imports, simple syntax problems, and basic accessibility issues.

Is ReactStream compatible with TypeScript?

Yes, ReactStream fully supports TypeScript components. It can analyze types, interfaces, and TypeScript-specific syntax to provide accurate feedback and suggestions.

How does ReactStream compare to other React tools?

ReactStream combines the functionality of multiple tools into one seamless workflow. While other tools might focus on just linting, testing, or component development, ReactStream provides a comprehensive solution for the entire component lifecycle.
