Documentation

Everything you need to know to get started with ReactStream.

Installation

# Install globally
npm install -g @reactstream/cli

# Or install in your project
npm install --save-dev @reactstream/cli

ReactStream requires Node.js v14.0.0 or higher and npm or yarn.

Commands

  • analyze

    Analyze components for issues and optimization opportunities.

  • serve

    Start a development server for component testing.

  • help

    Display help information and available commands.
